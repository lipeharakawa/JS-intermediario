Declaração de variáveis

Definição

var

Permite alterar os dados ou acessar em qualquer bloco, permite ser criada vazia.

var texto = "Oi";
var num = 2;

if(num > 1 ){
var texto = "Oi, tudo bem ?"
}

document.write(texto);

let

Permite somente alterar os dados ou acessar dentro do próprio bloco criado, permite ser criada vazia.

let texto = "Oi";
let num = 2;

if(num > 1 ){
let texto = "Oi, tudo bem ?"
document.write(texto);
}

const

Permite somente acessar os dados dentro do próprio bloco criado e não permite ser alterado em nenhum local, não permite ser criada vazia.

const cores = {
vermelho: "vermelho",
verde: "verde",
};

document.write(cores.vermelho);

cores.vermelho = "vermelho escuro";

document.write(cores.vermelho);
