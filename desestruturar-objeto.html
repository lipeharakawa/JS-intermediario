Desconstruindo objetos

Definição

Ao desestruturar (destructuring) um objeto, estamos quebrando uma estrutura complexa em fragmentos, criando partes mais simples de objetos.

let pessoa = {
nome:"Matheus",
sobrenome: "Fraga",
empresa: "Sujeito Programador",
cargo:"Programador FullStack"
};

let { nome:nomePessoa, cargo } = pessoa;

const aluno = {
nome : "Matheus",
idade : "23"
};

const {nome, idade} = aluno;
