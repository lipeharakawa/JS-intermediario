Temporizadores

Definição

Temporizadores são funções embutidas no JavaScript (built-in timer functions) para chamar funções de retorno após um determinado tempo.

setInterval

setInterval(nomeFuncao, tempo) significa que a função "nomeFuncao" será executada assim cada vez que se atingir o determinado tempo. O tempo padrão é dado por milésimos por segundo, ou seja, 1 segundo equivale a 1000 milésimos.

function acao(){
document.write("Executando... ");
}

setInterval(acao, 1000);

setTimeout

setTimeout(nomeFuncao, tempo) serve para executar uma função ou instrução quando atingir o seu tempo e essa instrução é executada apenas uma vez.

function acao(){
document.write("Executando... ");
}

setTimeout(acao, 1000);

Parar a função

var timer = setInterval(nomeFuncao, tempo);
clearInterval(timer);

var timer = setTimeout(nomeFuncao, tempo);
clearTimeout(timer);
