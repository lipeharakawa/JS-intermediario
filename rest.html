Rest Operator

Definição

Permite representar um número indefinido de argumentos na forma de um Array.

Exemplo 1

function convidados(...nomes){
document.write(nomes);
}
convidados("Matheus", "Lucas", "Maria");

Exemplo 2

function sorteador(...numeros){
const numeroGerado = Math.floor(Math.random() * numeros.length);
document.write("Número gerado foi: " + numeros[numeroGerado]);
}
sorteador(1, 4, 5, 15, 25, 90, 55, 34);
